CCSV

Description

The CCSV class is a simple class that is used to compose and decompose CSV strings. A CSV string is a string that contains comma separated value strings (otherwise known as elements). a CSV string looks like this:
Eat, Your, Wheaties

This is a SCV string composed of three elements. If any of the elements contain an imbedded comma, a double quote or significant leading or trailing whitespace, the entire element is double quoted. And in the case of inbedded double quotes, each double quote is doubled up. So a CSV line could look like this:
"Bart's pet poodle?", "Bart says ""Bring em on!"""
